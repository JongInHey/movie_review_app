## Movie Review app | 영화 리뷰 앱

### 이 앱을 만들어본 이유

- 영화를 좋아하는데 어떠한 영화를 봐야할 때 개별로 검색을 하거나 현재 상영중인 영화를 기준으로 찾았었습니다.

- 사용자의 편의성을 위해 보고 싶은 영화 또는 정보를 알고 싶은 영화의 내용들을 보다 쉽게 찾고 편리하게 보기 위해 만들게 되었습니다.

### Stack (사용된 기술)

- React
- javaScript
- node.js
- node-fetch
- html / css
- swiper
- git
- github
